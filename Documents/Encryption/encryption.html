Step-by-Step Guide to Encrypting Files Using StatiCrypt

1. Hosting on GitHub Pages

If you would like to create an encrypted webpage to be hosted on MBCIU GitHub Pages, contact Warda at wtsyeda@unimelb.edu.au. Alternately, you can use the following instructions to encrypt files using StatiCrypt.

2. What is StatiCrypt?

StatiCrypt is a tool that encrypts your static HTML files using AES-256. It generates a password-protected HTML file that users can decrypt in their browser.

3. Web-Based StatiCrypt

If you prefer not to use the command line, you can use the web-based version of StatiCrypt:

StatiCrypt Web App

Steps to encrypt a file using the web version:

  • Open the StatiCrypt web app.
  • Paste your HTML content into the provided text area.
  • Enter a strong password.
  • Click "Encrypt" to generate a secure HTML file.
  • Download and share the encrypted file as needed.

4. Install StatiCrypt (Command Line)

You can install StatiCrypt using npm. Open a terminal and run:

npm install -g staticrypt

5. Encrypt an HTML File

To encrypt a file (e.g., myfile.html), use the following command:

staticrypt myfile.html mySecretPassword

This will generate an encrypted HTML file, typically named myfile_encrypted.html, which prompts users to enter a password.

6. Customize Encryption Options

StatiCrypt provides options to customize the encryption process. For example, to change the output filename, use:

staticrypt myfile.html mySecretPassword -o encrypted.html

7. Decrypting the File

To decrypt the file, simply open it in a browser. StatiCrypt will prompt you for a password. Enter the password used during encryption to access the original content.

8. Additional Options

  • --title "Custom Title" - Set a custom title for the encrypted file.
  • --template "template.html" - Use a custom HTML template.
  • --password-input "Enter Password" - Customize the password prompt.

9. Example Use Case

If you have a private HTML file with sensitive information, encrypt it with:

staticrypt private.html superSecure123

Then share the private_encrypted.html file with the intended recipient. They only need a browser and the correct password to view it.

10. Conclusion

StatiCrypt is a simple and effective way to protect static HTML content. By encrypting sensitive files, you ensure data security without relying on external services.
